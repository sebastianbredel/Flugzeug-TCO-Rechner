# Flugzeug-TCO-Rechner
Der Rechner zeigt dir auf Basis deiner Flugstunden und der Personenzahl einer Haltergemeinschaft, ob ein eigenes Ultraleichtflugzeug oder die Mitgliedschaft in Verein bzw. Flugschule die günstigere Lösung ist – inklusive aller Fix-, Betriebs- und Kapitalkosten und dem Break-even-Punkt.
